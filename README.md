# ByteForge

A modern front-end web app built with **React + Vite**.

## What this project is
ByteForge is a lightweight React application scaffold intended for building a fast UI with a modern dev workflow (HMR, linting, production builds).

## Tech stack
- React
- Vite
- JavaScript
- ESLint

## Project structure
- `src/` — application source code
- `public/` — static assets

## Getting started
### Prerequisites
- Node.js (LTS recommended)
- npm

### Install
```bash
npm install
```

### Run (dev)
```bash
npm run dev
```
The dev server will print a local URL (commonly `http://localhost:5173`).

### Build
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Scripts
- `npm run dev` — start dev server
- `npm run build` — build for production
- `npm run preview` — preview the production build
- `npm run lint` — run ESLint (if configured)
